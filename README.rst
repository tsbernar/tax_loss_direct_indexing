========
tax_loss
========

Development
--------
Getting started:
1. conda create -n tax_loss python=3.9 poetry invoke -y  # Create conda env
2. conda activate tax_loss  # Activate env
3. inv update  # Install dependencies
4. inv install # Install package

Running locally:
1. poetry run tax_loss

Building
--------
1. inv build
